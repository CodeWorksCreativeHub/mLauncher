# Forked com recursos extras

- Removida a confusão, como anúncios e links
- É possível renomear aplicações na gaveta da aplicação (a renomeação de aplicações no ecrã inicial já é suportada. Basta um clique longo numa aplicação no ecrã inicial e começar a digitar)
- Muito mais personalizabilidade nos gestos do ecrã inicial
    - Os gestos estão agora:
        - Deslizando para cima, para baixo, esquerda, direita
        - Clicando no relógio
        - Clicando na Data
    - Possíveis ações agora incluem:
        - Abrir o aplicativo especificado
        - Exibir lista de aplicativos
        - Bloqueando a tela
        - Mostrar aplicativos abertos recentemente
        - Abrindo a gaveta de notificação
        - Abrir as configurações rápidas
        - Diálogo de energia
        - Faça uma captura de tela
        - Desativar gesto
- Alinhar o relógio independentemente das aplicações domésticas
- Alterar o alinhamento das aplicações no app-drawer
- Bloquear configurações por trás da biometria (alternar)
