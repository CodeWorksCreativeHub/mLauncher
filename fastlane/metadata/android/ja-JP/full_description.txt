<h1>マルチランチャー: 読みやすさと操作性を重視して設計された、シンプルでクリーンな Android ランチャー</h1>

<p>煩雑さを省いて簡単に使えるランチャーをお探しですか? <strong>マルチランチャー</strong> は、小さなアイコンを明確で読みやすいテキストボタンに置き換え、携帯電話の操作を容易にする、強力でありながらミニマリストな Android ランチャーです。</p>

<p>シンプルなホーム画面が必要な場合でも、視認性を高めるために大きなテキストが必要な場合でも、ジェスチャーベースの素早いアクションを好む場合でも、Multi Launcher を使用すると、オプションを多用することなく、完全な制御が可能になります。</p>

<h2>マルチランチャーを選ぶ理由</h2>
<ul>
    <li><strong>テキストベースのランチャー:</strong> わかりにくいアイコンはありません。アプリにはラベル付きのテキスト ボタンだけが表示されます。</li>
    <li><strong>高度なカスタマイズが可能:</strong> レイアウト、フォント サイズ、時計の位置などを調整します。</li>
    <li><strong>クイック ジェスチャ:</strong> タップ、スワイプ、ロング スワイプを使用して、アプリを開いたり、画面をロックしたり、通知にアクセスしたりします。</li>
    <li><strong>安全:</strong> 設定とプライバシーのための生体認証ロック。</li>
    <li><strong>高齢者やシンプルさを好む方に最適:</strong> 読みやすく、使いやすく、カスタマイズしやすい。</li>
</ul>

<h2>主な特徴:</h2>
<ul>
    <li>カスタム アプリ名: アプリの名前を、自分のイメージに合うように変更します。</li>
    <li>柔軟なレイアウト: 自分に合った方法でアプリを整理します。</li>
    <li>時計と日付のカスタマイズ: ウィジェットを好きな場所に配置できます。</li>
    <li>アプリ使用状況分析: 各アプリに費やした時間を確認します。</li>
    <li>ジェスチャーショートカット: ダブルタップ、スワイプなどのカスタムアクションを設定します。</li>
</ul>

<h2>アクセシビリティ機能:</h2>
<p>このアプリは <strong>AccessibilityService API</strong> を使用して次のことを行います:</p>
<ul>
    <li>ジェスチャーで画面をロックします</li>
    <li>電源メニューを開く</li>
    <li>通知ドロワーを下に引いてください</li>
</ul>
