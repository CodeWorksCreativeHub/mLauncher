<h2>追加機能付きフォーク</h2>

<ul>
    <li>広告やリンクなどの不要なものを削除しました。</li>
    <li>アプリドロワー内のアプリの名前を変更できます (ホーム画面上のアプリの名前変更はすでにサポートされています。 ホーム画面でアプリを長押しして入力を開始するだけです。)</li> ホーム画面でアプリを長押しして入力を開始するだけです。)</li>
    <li>ホーム画面のジェスチャのオプションをさらに追加しました:
        <ul>
            <li><b>ジェスチャーに以下が含まれるようになりました:</b>
                <ul>
                    <li>ダブルタップ</li>
                    <li>時計をクリック</li>
                    <li>日付をクリック</li>
                    <li>合計使用量をクリック</li>
                    <li>上に短くスワイプ</li>
                    <li>下に短くスワイプ</li>
                    <li>左に短くスワイプ</li>
                    <li>右に短くスワイプ</li>
                    <li>上に長くスワイプ</li>
                    <li>下に長くスワイプ</li>
                    <li>左に長くスワイプ</li>
                    <li>右に長くスワイプ</li>
                </ul>
            </li>
            <li><b>実行可能なアクションは次のとおりです:</b>
                <ul>
                    <li>指定したアプリを開く</li>
                    <li>アプリリストを表示</li>
                    <li>画面をロックします</li>
                    <li>最近開いたアプリを表示</li>
                    <li>通知ドロワーを開く</li>
                    <li>クイック設定を開く</li>
                    <li>パワーダイアログ</li>
                    <li>スクリーンショットを撮る</li>
                    <li>ページ切り替え</li>
                    <li>ジェスチャーを無効にする</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>ホームアプリとは独立して時計を配置することもできます。</li>
    <li>ホーム画面とは独立して、アプリドロワー内のアプリの配置を変更します。</li>
    <li>ホーム画面のセクションのフォントサイズを個別に変更します。</li>
    <li>インターネット権限を削除しました。 アプリ開発者があなたについて何を知りたいかはわかりません。</li>
    <li>生体認証による設定をロックします（トグル）。</li>
    <li>クラッシュを電子メールで報告する機能を追加しました。</li>
    <li>アプリの使用時間を表示する機能を追加しました。</li>
</ul>
