<p>Bifurcado de <a href="https://github.com/tanujnotes/Olauncher">Olauncher</a>. Minimalista y limpio.</p>

<h2>Bifurcado con funciones adicionales</h2>

<ul>
    <li>Se eliminaron elementos innecesarios, como anuncios y enlaces.</li>
    <li>Puedes cambiar el nombre de las aplicaciones en el cajón de aplicaciones (ya se admite cambiar el nombre de las aplicaciones en la pantalla de inicio). Simplemente haz clic prolongado en una aplicación en la pantalla de inicio y comienza a escribir).</li>
    <li>Hemos agregado muchas más opciones para gestos en la pantalla de inicio:
        <ul>
            <li><b>Los gestos ahora incluyen:</b>
                <ul>
                    <li>Doble toque</li>
                    <li>Haciendo clic en el reloj</li>
                    <li>Haciendo clic en la fecha</li>
                    <li>Haciendo clic en el uso total</li>
                    <li>Desliza brevemente hacia arriba</li>
                    <li>Desliza brevemente hacia abajo</li>
                    <li>Desliza brevemente hacia la izquierda</li>
                    <li>Desliza brevemente hacia la derecha</li>
                    <li>Desliza el dedo hacia arriba durante un rato</li>
                    <li>Desliza el dedo hacia abajo durante un rato</li>
                    <li>Deslizamiento largo hacia la izquierda</li>
                    <li>Desliza el dedo hacia la derecha durante un rato</li>
                </ul>
            </li>
            <li><b>Las posibles acciones ahora incluyen:</b>
                <ul>
                    <li>Abrir la aplicación especificada</li>
                    <li>Mostrar lista de aplicaciones</li>
                    <li>Bloquear la pantalla</li>
                    <li>Mostrar aplicaciones abiertas recientemente</li>
                    <li>Abre el cajón de notificaciones</li>
                    <li>Abrir configuración rápida</li>
                    <li>Diálogo de energía</li>
                    <li>Tomar una captura de pantalla</li>
                    <li>Cambiar página</li>
                    <li>Deshabilitar gesto</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>También puedes colocar el reloj independientemente de las aplicaciones de inicio.</li>
    <li>Cambia la alineación de las aplicaciones en el cajón de aplicaciones independientemente de la pantalla de inicio.</li>
    <li>Cambia el tamaño de fuente de las secciones de la pantalla de inicio de forma independiente una de otra.</li>
    <li>Se eliminó el permiso de Internet. Nunca se sabe lo que un desarrollador de aplicaciones quiere saber sobre ti.</li>
    <li>Bloquear ajustes detrás de datos biométricos (activar/desactivar).</li>
    <li>Se agregó la capacidad de informar fallas por correo electrónico.</li>
    <li>Se agregó la capacidad de ver el tiempo de uso de la aplicación.</li>
</ul>
