<h2>Bifurcado con funciones adicionales</h2>

<b>Características:</b>

- Personalización del cajón de aplicaciones: cambie el nombre de las aplicaciones en el cajón de aplicaciones (el cambio de nombre de las aplicaciones de la pantalla de inicio ya es compatible con una aplicación larga).
hacer clic).
- Gestos avanzados: una amplia gama de gestos, incluidos toques y deslizamientos, para activar diferentes acciones.
- Personalización del reloj: coloca el reloj independientemente de las aplicaciones de inicio, lo que te brinda más control sobre tu pantalla de inicio.
disposición.
- Diseño del cajón de aplicaciones: cambia la alineación de las aplicaciones en el cajón de aplicaciones independientemente de la pantalla de inicio.
- Personalización del tamaño de fuente: ajuste el tamaño de fuente de las secciones de la pantalla de inicio por separado.
- Centrado en la privacidad: se eliminó el permiso de Internet, lo que garantiza que sus datos permanezcan privados.
- Seguridad biométrica: Bloquear configuraciones detrás de la autenticación biométrica (opción de alternancia).
- Informes de fallos: capacidad de informar fallos por correo electrónico para una mejor resolución de problemas.
- Seguimiento del uso de la aplicación: vea el tiempo de uso de la aplicación directamente desde la pantalla de inicio.

<b>Los gestos incluyen:</b>

- Doble toque: toca rápidamente dos veces para activar una acción predefinida.
- Toque el reloj: toque el reloj para acceder a funciones o configuraciones adicionales.
- Toque de fecha: toque la fecha para acceder fácilmente al calendario.
- Toque de uso: toque el uso total para ver los datos de uso del sistema o de la aplicación.
- Deslizamientos cortos: deslice hacia arriba, abajo, izquierda o derecha para navegar o activar diferentes acciones.
- Deslizamientos largos: deslice hacia arriba, abajo, izquierda o derecha para realizar acciones gestuales adicionales.

<b>Las posibles acciones incluyen:</b>

- Abrir aplicación específica: abre instantáneamente cualquier aplicación que elijas con un gesto.
- Mostrar lista de aplicaciones: muestra la lista completa de aplicaciones instaladas en el dispositivo.
- Pantalla de bloqueo: bloquea tu dispositivo por seguridad.
- Mostrar aplicaciones abiertas recientemente: acceda a las aplicaciones utilizadas recientemente para realizar múltiples tareas rápidamente.
- Abrir cajón de notificaciones: activa el cajón de notificaciones deslizando el dedo o tocando.
- Abrir configuración rápida: accede a la configuración rápida directamente desde la pantalla de inicio.
- Cuadro de diálogo de energía: abre opciones de energía como apagar, reiniciar y más.
- Tomar una captura de pantalla: captura la pantalla para guardar o compartir tu contenido.
- Cambiar página: cambia rápidamente entre las páginas de la pantalla de inicio.
- Deshabilitar gesto: deshabilite gestos específicos según sea necesario.
