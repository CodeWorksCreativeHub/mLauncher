<p> <a href="https://github.com/tanujnotes/Olauncher">Olauncher</a>에서 포크되었습니다. 최소한이고 깔끔합니다.</p>

<h2>추가 기능이 포함된 포크</h2>

<ul>
    <li>광고와 링크 등의 불필요한 내용을 제거했습니다.</li>
    <li>앱 서랍에서 앱 이름을 바꿀 수 있습니다. (홈 화면에서 앱 이름을 바꾸는 것은 이미 지원됩니다.) 홈 화면에서 앱을 길게 클릭하고 입력을 시작하세요.)</li>
    <li>홈 화면 제스처에 대한 옵션을 더 많이 추가했습니다.
        <ul>
            <li><b>제스처에 다음이 추가되었습니다:</b>
                <ul>
                    <li>두 번 탭</li>
                    <li>시계 클릭</li>
                    <li>날짜를 클릭하세요</li>
                    <li>전체 사용량 클릭</li>
                    <li>짧게 위로 스와이프</li>
                    <li>짧게 아래로 스와이프</li>
                    <li>왼쪽으로 짧게 스와이프</li>
                    <li>오른쪽으로 짧게 스와이프</li>
                    <li>위로 길게 스와이프</li>
                    <li>아래로 길게 스와이프</li>
                    <li>왼쪽으로 길게 스와이프</li>
                    <li>오른쪽으로 길게 스와이프</li>
                </ul>
            </li>
            <li><b>이제 가능한 작업은 다음과 같습니다.</b>
                <ul>
                    <li>지정된 앱 열기</li>
                    <li>앱 목록 표시</li>
                    <li>화면 잠금</li>
                    <li>최근에 열었던 앱 표시</li>
                    <li>알림 서랍을 엽니다</li>
                    <li>빠른 설정 열기</li>
                    <li>전원 대화 상자</li>
                    <li>스크린샷 찍기</li>
                    <li>페이지 전환</li>
                    <li>제스처 비활성화</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>홈 앱과 상관없이 시계 위치를 조정할 수도 있습니다.</li>
    <li>홈 화면과 별도로 앱 서랍에서 앱의 정렬을 변경합니다.</li>
    <li>홈 화면 섹션의 글꼴 크기를 서로 독립적으로 변경합니다.</li>
    <li>인터넷 권한이 제거되었습니다. 앱 개발자가 당신에 대해 무엇을 알고 싶어하는지 알 수 없습니다.</li>
    <li>생체 인식 뒤의 잠금 설정(토글).</li>
    <li>이메일을 통해 충돌을 보고하는 기능을 추가했습니다.</li>
    <li>앱 사용 시간을 볼 수 있는 기능이 추가되었습니다.</li>
</ul>
