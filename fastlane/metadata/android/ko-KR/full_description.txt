- 시계를 클릭할 때 앱이나 동작을 변경하는 기능
- 날짜 클릭 시 앱 또는 동작 변경 기능
- 위, 아래, 왼쪽 또는 오른쪽으로 스와이프할 때 앱 또는 동작을 변경하는 기능
- 홈 앱과 독립적으로 시계 및 날짜 정렬
- 앱 서랍에서 앱 정렬 변경
