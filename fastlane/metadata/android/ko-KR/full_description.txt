<h1>멀티 런처: 가독성과 제어력을 위해 설계된 깔끔하고 심플한 안드로이드 런처</h1>

<p>복잡함을 없애고 간편하게 사용할 수 있는 런처를 찾고 계신가요? <strong>멀티 런처</strong> 는 작았던 아이콘을 선명하고 읽기 쉬운 텍스트 버튼으로 대체하여 휴대폰 탐색을 더욱 편리하게 해주는 강력하면서도 미니멀한 안드로이드 런처입니다.</p>

<p>심플한 홈 화면을 원하든, 시인성을 위해 큰 텍스트가 필요하든, 빠른 제스처 기반 동작을 선호하든, Multi Launcher는 복잡한 옵션 없이 완벽한 제어 기능을 제공합니다.</p>

<h2>멀티 런처를 선택하는 이유는 무엇일까요?</h2>
<ul>
    <li><strong>텍스트 기반 런처:</strong> 복잡한 아이콘 없이 앱에 대한 레이블이 지정된 텍스트 버튼만 있습니다.</li>
    <li><strong>고도로 맞춤 설정 가능:</strong> 레이아웃, 글꼴 크기, 시계 위치 등을 조정할 수 있습니다.</li>
    <li><strong>빠른 제스처:</strong> 탭, 스와이프, 길게 스와이프하여 앱을 열고, 화면을 잠그고, 알림을 확인하세요.</li>
    <li><strong>보안:</strong> 설정 및 개인 정보 보호를 위한 생체 인식 잠금.</li>
    <li><strong>어르신이나 심플함을 좋아하시는 분들께 안성맞춤:</strong> 읽기 쉽고, 사용하기 편하며, 맞춤 설정도 간편합니다.</li>
</ul>

<h2>주요 특징:</h2>
<ul>
    <li>사용자 지정 앱 이름: 앱 이름을 원하는 대로 변경하세요.</li>
    <li>유연한 레이아웃: 자신에게 맞는 방식으로 앱을 구성하세요.</li>
    <li>시계 및 날짜 사용자 지정: 원하는 위치에 위젯을 배치하세요.</li>
    <li>앱 사용 통계: 각 앱에서 얼마나 많은 시간을 보내는지 확인하세요.</li>
    <li>제스처 바로가기: 두 번 탭, 스와이프 등에 대한 사용자 지정 동작을 설정하세요.</li>
</ul>

<h2>접근성 기능:</h2>
<p>이 앱은 <strong>AccessibilityService API</strong> 를 사용하여 다음을 수행합니다.</p>
<ul>
    <li>제스처로 화면 잠금</li>
    <li>전원 메뉴를 엽니다</li>
    <li>알림 창을 아래로 당기세요</li>
</ul>

<p><em>멀티 런처는 사용자의 개인정보를 존중합니다. 이러한 기능은 개인 데이터를 수집하거나 공유하지 않습니다.</em></p>

<p>화려하고 복잡한 런처에 지쳐 간단하고, 세련되고, 강력한 기능을 원하신다면 <strong>멀티 런처</strong> 가 완벽한 홈 화면 솔루션입니다.</p>
