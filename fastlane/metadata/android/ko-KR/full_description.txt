# 추가 기능이 포함된 포크

- 광고, 링크 등의 불필요한 내용을 제거했습니다.
- 앱 서랍에서 앱 이름을 변경할 수 있습니다(홈 화면에서 앱 이름을 변경하는 것은 이미 지원됩니다. 홈 화면에서 앱을 길게 클릭하고 입력을 시작하세요.)
- 홈 화면 제스처에 대한 옵션이 훨씬 더 추가되었습니다.
    - 제스처는 다음과 같습니다.
        - 위, 아래, 왼쪽, 오른쪽으로 스와이프
        - 시계를 클릭한다
        - 날짜를 클릭
    - 이제 가능한 작업은 다음과 같습니다.
        - 지정된 앱 열기
        - 앱 목록 표시
        - 화면 잠금
        - 최근 열었던 앱 표시
        - 알림 서랍 열기
        - 빠른 설정 열기
        - 파워 다이얼로그
        - 스크린샷 찍기
        - 제스처 비활성화
- 홈 앱과 상관없이 시계의 위치를 조정할 수 있습니다.
- 홈 화면과 별도로 앱 서랍에서 앱 정렬을 변경합니다.
- 홈 화면의 섹션별 글꼴 크기를 서로 독립적으로 변경합니다.
- 인터넷 권한이 제거되었습니다. 앱 개발자가 당신에 대해 무엇을 알고 싶어하는지 알 수 없습니다.
- 생체 인식 뒤의 잠금 설정(토글)
- 이메일을 통해 충돌을 보고하는 기능이 추가되었습니다.
- 앱 사용시간을 볼 수 있는 기능이 추가되었습니다.
