# Forked dengan fitur tambahan

- Menghapus kekacauan, seperti iklan dan tautan.
- Anda dapat mengganti nama aplikasi di app-drawer (Mengganti nama aplikasi di layar beranda sudah didukung. Cukup klik lama aplikasi di layar beranda dan mulailah mengetik.)
- Kami telah menambahkan lebih banyak opsi untuk gerakan di layar beranda:
    - Gerakan sekarang:
        - Menggesek ke atas, bawah, kiri, kanan
        - Mengklik pada jam
        - Mengklik Tanggal
    - Kemungkinan tindakan sekarang meliputi:
        - Buka aplikasi tertentu
        - Menampilkan daftar aplikasi
        - Mengunci layar
        - Tampilkan aplikasi yang baru dibuka
        - Membuka laci notifikasi
        - Membuka pengaturan cepat
        - Dialog Kekuatan
        - Ambil Tangkapan Layar
        - Nonaktifkan Gerakan
- Anda juga dapat memposisikan jam secara terpisah dari aplikasi rumah.
- Ubah perataan aplikasi di laci aplikasi secara independen dari layar beranda.
- Ubah ukuran font bagian layar beranda secara independen satu sama lain.
- Izin internet dihapus. Anda tidak pernah tahu apa yang ingin diketahui oleh pengembang aplikasi tentang Anda.
- Pengaturan kunci di belakang biometrik (alihkan)
- Menambahkan kemampuan untuk melaporkan kerusakan melalui email.
- Menambahkan kemampuan untuk melihat waktu penggunaan aplikasi.
