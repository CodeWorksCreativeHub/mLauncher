<p>Abgezweigt von <a href="https://github.com/tanujnotes/Olauncher">Olauncher</a>: Ein minimaler und sauberer Launcher mit
    erweiterte Funktionen.</p>

<h2>Mit zusätzlichen Funktionen gegabelt</h2>

<b>Funktionen:</b>

- App-Schubladenanpassung: Benennen Sie Apps in der App-Schublade um (die Umbenennung von Apps auf dem Home-Bildschirm wird bereits mit einer langen
klicken).
- Erweiterte Gesten: Eine große Auswahl an Gesten, einschließlich Tippen und Wischen, um verschiedene Aktionen auszulösen.
- Uhranpassung: Positionieren Sie die Uhr unabhängig von den Home-Apps und haben Sie so mehr Kontrolle über Ihren Home-Bildschirm
Layout.
- App-Drawer-Layout: Ändern Sie die Ausrichtung von Apps im App-Drawer unabhängig vom Home-Bildschirm.
- Anpassung der Schriftgröße: Passen Sie die Schriftgröße einzelner Abschnitte des Startbildschirms separat an.
- Auf Datenschutz ausgerichtet: Internetberechtigung entfernt, um sicherzustellen, dass Ihre Daten privat bleiben.
- Biometrische Sicherheit: Sperreinstellungen hinter biometrischer Authentifizierung (Umschaltoption).
- Absturzberichte: Möglichkeit, Abstürze zur besseren Fehlerbehebung per E-Mail zu melden.
- App-Nutzungsverfolgung: Zeigen Sie die App-Nutzungszeit direkt vom Startbildschirm aus an.

<b>Gesten umfassen:</b>

- Doppeltippen: Tippen Sie schnell zweimal, um eine vordefinierte Aktion auszulösen.
- Tippen Sie auf die Uhr: Tippen Sie auf die Uhr, um auf zusätzliche Funktionen oder Einstellungen zuzugreifen.
- Auf Datum tippen: Tippen Sie auf das Datum, um einfach auf den Kalender zuzugreifen.
- Tippen Sie auf die Nutzung: Tippen Sie auf die Gesamtnutzung, um Daten zur System- oder App-Nutzung anzuzeigen.
- Kurze Wischbewegungen: Wischen Sie nach oben, unten, links oder rechts, um zu navigieren oder verschiedene Aktionen auszulösen.
- Lange Wischbewegungen: Wischen Sie für zusätzliche Gestenaktionen nach oben, unten, links oder rechts.

<b>Mögliche Aktionen umfassen:</b>

- Bestimmte App öffnen: Öffnen Sie mit einer Geste sofort eine App Ihrer Wahl.
- App-Liste anzeigen: Zeigt die vollständige Liste der auf dem Gerät installierten Apps an.
- Sperrbildschirm: Sperren Sie Ihr Gerät aus Sicherheitsgründen.
- Zuletzt geöffnete Apps anzeigen: Greifen Sie für schnelles Multitasking auf zuletzt verwendete Apps zu.
- Benachrichtigungsleiste öffnen: Lösen Sie die Benachrichtigungsleiste durch Wischen oder Tippen aus.
- Schnelleinstellungen öffnen: Greifen Sie direkt vom Startbildschirm auf die Schnelleinstellungen zu.
- Energiedialogfeld: Öffnen Sie Energieoptionen wie Herunterfahren, Neustart und mehr.
- Machen Sie einen Screenshot: Erfassen Sie den Bildschirm, um Ihre Inhalte zu speichern oder freizugeben.
- Seite wechseln: Schnell zwischen den Seiten des Startbildschirms wechseln.
- Gesten deaktivieren: Deaktivieren Sie bestimmte Gesten nach Bedarf.
