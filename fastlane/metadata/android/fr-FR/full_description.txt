Fourchu à partir de <a href="https://github.com/tanujnotes/Olauncher">Olauncher</a>. Minimal et propre.

# Fork avec des fonctionnalités supplémentaires

- Suppression de l'encombrement, comme les publicités et les liens.
- Vous pouvez renommer des applications dans le tiroir d'applications (le changement de nom des applications sur l'écran d'accueil est déjà pris en charge. (Cliquez simplement longuement sur une application sur l'écran d'accueil et commencez à taper.)
- Nous avons ajouté beaucoup plus d'options pour les gestes sur l'écran d'accueil :
    - Les gestes sont désormais :
        - Glisser vers le haut, le bas, la gauche, la droite
        - Cliquer sur l'horloge
        - En cliquant sur la date
    - Les actions possibles incluent désormais :
        - Ouvrir l'application spécifiée
        - Afficher la liste des applications
        - Verrouillage de l'écran
        - Afficher les applications récemment ouvertes
        - Ouverture du tiroir de notifications
        - Ouverture des paramètres rapides
        - Dialogue de puissance
        - Prendre une capture d'écran
        - Désactiver le geste
- Vous pouvez également positionner l'horloge indépendamment des applications domestiques.
- Modifiez l'alignement des applications dans le tiroir d'applications indépendamment de l'écran d'accueil.
- Modifiez la taille de la police des sections de l'écran d'accueil indépendamment les unes des autres.
- Suppression de l'autorisation Internet. Vous ne savez jamais ce qu’un développeur d’applications veut savoir sur vous.
- Verrouiller les paramètres derrière la biométrie (bascule)
- Ajout de la possibilité de signaler un crash par e-mail.
- Ajout de la possibilité d'afficher le temps d'utilisation de l'application.
