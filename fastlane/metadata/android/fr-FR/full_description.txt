<h2>Fourchu avec des fonctionnalités supplémentaires</h2>

<b>Caractéristiques :</b>

- Personnalisation du tiroir d'applications : renommer les applications dans le tiroir d'applications (le renommage des applications sur l'écran d'accueil est déjà pris en charge avec un long
cliquez).
- Gestes avancés : une large gamme de gestes, notamment des appuis et des balayages, pour déclencher différentes actions.
- Personnalisation de l'horloge : positionnez l'horloge indépendamment des applications d'accueil, ce qui vous donne plus de contrôle sur votre écran d'accueil
mise en page.
- Disposition du tiroir d'applications : modifiez l'alignement des applications dans le tiroir d'applications indépendamment de l'écran d'accueil.
- Personnalisation de la taille de la police : ajustez la taille de la police des sections de l'écran d'accueil séparément.
- Axé sur la confidentialité : suppression de l'autorisation Internet, garantissant que vos données restent privées.
- Sécurité biométrique : verrouiller les paramètres derrière l'authentification biométrique (option de basculement).
- Rapports de plantage : possibilité de signaler les plantages par e-mail pour un meilleur dépannage.
- Suivi de l'utilisation de l'application : affichez le temps d'utilisation de l'application directement depuis l'écran d'accueil.

<b>Les gestes incluent :</b>

- Double Tap : appuyez rapidement deux fois pour déclencher une action prédéfinie.
- Horloge Tap : appuyez sur l'horloge pour accéder à des fonctionnalités ou paramètres supplémentaires.
- Date Tap : appuyez sur la date pour accéder facilement au calendrier.
- Utilisation : appuyez sur l'utilisation totale pour afficher les données d'utilisation du système ou de l'application.
- Balayages courts : balayez vers le haut, le bas, la gauche ou la droite pour naviguer ou déclencher différentes actions.
- Balayages longs : balayez vers le haut, le bas, la gauche ou la droite pour des actions gestuelles supplémentaires.

<b>Les actions possibles incluent :</b>

- Ouvrir l'application spécifiée : ouvrez instantanément n'importe quelle application de votre choix d'un geste.
- Afficher la liste des applications : affiche la liste complète des applications installées sur l'appareil.
- Écran de verrouillage : verrouillez votre appareil pour plus de sécurité.
- Afficher les applications récemment ouvertes : accédez aux applications récemment utilisées pour un multitâche rapide.
- Ouvrir le tiroir de notifications : déclenchez le tiroir de notifications avec un glissement ou un appui.
- Ouvrir les paramètres rapides : accédez aux paramètres rapides directement depuis l'écran d'accueil.
- Boîte de dialogue d'alimentation : ouvrez les options d'alimentation telles que l'arrêt, le redémarrage, etc.
- Prendre une capture d'écran : capturez l'écran pour enregistrer ou partager votre contenu.
- Changer de page : basculez rapidement entre les pages de l'écran d'accueil.
- Désactiver le geste : désactivez des gestes spécifiques selon vos besoins.
