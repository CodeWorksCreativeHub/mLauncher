<p>Fourchu à partir de <a href="https://github.com/tanujnotes/Olauncher">Olauncher</a>. Minimaliste et propre.</p>

<h2>Fourchu avec des fonctionnalités supplémentaires</h2>

<ul>
    <li>Suppression des éléments encombrants, comme les publicités et les liens.</li>
    <li>Vous pouvez renommer des applications dans le tiroir d'applications (le renommage des applications sur l'écran d'accueil est déjà pris en charge. Cliquez longuement sur une application sur l'écran d'accueil et commencez à taper.)</li>
    <li>Nous avons ajouté beaucoup plus d'options pour les gestes sur l'écran d'accueil :
        <ul>
            <li><b>Les gestes incluent désormais :</b>
                <ul>
                    <li>Appuyez deux fois</li>
                    <li>Cliquer sur l'horloge</li>
                    <li>En cliquant sur la date</li>
                    <li>En cliquant sur l'utilisation totale</li>
                    <li>Glissement court vers le haut</li>
                    <li>Glissement court vers le bas</li>
                    <li>Glissement court vers la gauche</li>
                    <li>Glissement court vers la droite</li>
                    <li>Balayage long vers le haut</li>
                    <li>Glissement long vers le bas</li>
                    <li>Balayage long vers la gauche</li>
                    <li>Balayage long vers la droite</li>
                </ul>
            </li>
            <li><b>Les actions possibles incluent désormais :</b>
                <ul>
                    <li>Ouvrir l'application spécifiée</li>
                    <li>Afficher la liste des applications</li>
                    <li>Verrouiller l'écran</li>
                    <li>Afficher les applications récemment ouvertes</li>
                    <li>Ouvrir le tiroir de notifications</li>
                    <li>Ouvrir les paramètres rapides</li>
                    <li>Dialogue d'alimentation</li>
                    <li>Prendre une capture d'écran</li>
                    <li>Changer de page</li>
                    <li>Désactiver le geste</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>Vous pouvez également positionner l'horloge indépendamment des applications domestiques.</li>
    <li>Modifiez l'alignement des applications dans le tiroir d'applications indépendamment de l'écran d'accueil.</li>
    <li>Modifiez la taille de la police des sections de l'écran d'accueil indépendamment les unes des autres.</li>
    <li>Autorisation Internet supprimée. On ne sait jamais ce qu'un développeur d'applications veut savoir sur vous.</li>
    <li>Verrouiller les paramètres derrière la biométrie (bascule).</li>
    <li>Ajout de la possibilité de signaler les plantages par e-mail.</li>
    <li>Ajout de la possibilité d'afficher le temps d'utilisation de l'application.</li>
</ul>
